## Цифровой прорыв 2020, компания «ITS»

Задача «Распознавание микроколоний в бактериальном посеве по бактериям» – [ссылка](https://cups.mail.ru/ru/tasks/1030).

В задаче давали 6 баллов за классификацию и один за сегментацию.

### Метрика соревнования

6 баллов за классификацию + 1 балл за сегментацию.

### Решение

Поскольку на одном снимке могли присутствовать только бактерии одного вида я решил разделить задачу мульти классовой сегментации бактерий на две под задачи:

- Классификацию снимков бактерий на по их видам. Для данной подзадачи я применил «Fine tuning» по методу «Differential LR Groups» для EfficientNet предобученной на ImageNet.
- Для каждого из 6 классов бактерий осуществлял бинарную сегментацию (есть ли данный тип бактерий и где попиксельно). Для бинарной сегментации я применял «Fine tuning» предобученого много стадийного детектора Masked-RCNN, «Fine tuning» сегментатора Unet сделанного из предобученного Resnet и EfficientNet, еще пробовал «Fine tuning» модели DeepLabV3.

Данный подход позволил мне классифицировать бактерии со 100% точностью даже на тестовом датасете, потому что бактерии и их питательная среда отличались друг от друга в том числе и по цвету (6 баллов).

Классификатор предсказывал классы всем снимкам из тестовой выборки. Проходя по циклу по всем картинкам по одной вызвалась модель для бинарной сегментации, ей загружались веса с которыми она была обучена предсказывать маску для данного класса, после чего модель предсказывала маску – [ссылка на Jupiter notebook с Masked-RCNN](https://colab.research.google.com/drive/1CTPmwjvsLD1Uvd8w5Lzefj8zpQpymzQ9?usp=sharing).

### Результат

37 место – [ссылка](https://cups.mail.ru/ru/results/leadersofdigital?page=2&pageSize=36&period=past&roundId=582)
